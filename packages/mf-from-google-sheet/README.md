# mf-from-sheet

[![NPM version][npm-image]][npm-url]
[![npm download][download-image]][download-url]

Retrieve and format molecular formula from google sheet.

## Installation

`$ npm install --save mf-from-google-sheet`

```js

const result = library(args);
```
