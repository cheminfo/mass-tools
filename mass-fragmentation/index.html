reactionFragmentation

Fragment a molecule by applying reactions from a custom database of reactions

reactionFragmentation(molecule: any, options: Object): object
Parameters
molecule (any)
options (Object = {})
Name Description
options.databaseName string (default 'cid') The database to be used
options.mode string (default 'positive') The mode to be used
options.maxDepth number (default 5) The maximum depth of the fragmentation tree
options.getProductsTrees boolean (default false) If true, the products trees are returned else products is an empty array
options.limitReactions number (default 200) The maximum number of reactions to be applied
options.maxIonizationDepth number (default 1) The maximum depth of the ionization tree
options.customDatabase Object (default {}) A custom database of reactions
options.customDatabase.positive Array? A custom database of reactions for positive mode
options.customDatabase.negative Array? A custom database of reactions for negative mode
options.customDatabase.ionization Object? A custom database ionization reactions
options.customDatabase.ionization.positive Array? A custom database of ionization reactions for positive mode
options.customDatabase.ionization.negative Array? A custom database of ionization reactions for negative mode
Returns
object: In-Silico fragmentation results with the following properties:
  • masses: array of monoisotopic masses
  • trees: array of fragmentation trees
  • products: array of trees grouped by product idCode
